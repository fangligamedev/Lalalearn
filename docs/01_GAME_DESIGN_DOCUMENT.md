## 1. 概述与愿景

### 1.1 产品定位

**Lalalearn** 是一个由 AI 驱动的通用游戏化学习平台，支持任意领域的知识培训。区别于传统学习软件，Lalalearn 采用：

- 🎮 **游戏化闯关机制** - 将枯燥的知识点转化为关卡挑战
- 🧠 **艾宾浩斯遗忘曲线** - 智能复习系统，确保长期记忆
- 🤖 **LLM Vibe 教授** - 情绪感知的 AI 导师，提供个性化辅导

### 1.2 目标用户

| 用户群体 | 使用场景 | 核心价值 |
|---------|---------|---------|
| 🧒 儿童/学生 | Python 编程入门、学科知识学习 | 趣味化、成就感 |
| 👨‍💼 企业员工 | 产品培训、技术文档学习 | 高效掌握、可追踪进度 |
| 📚 自学者 | 任意领域知识学习 | 定制化、自主节奏 |
| 🏢 培训机构 | 课程内容游戏化 | 提升学员参与度 |

### 5.2 复习调度算法

```typescript
interface SpacedRepetition {
  // 每个知识点的复习间隔 (小时)
  intervals: [1, 6, 24, 72, 168, 336, 720]; // 1h, 6h, 1d, 3d, 7d, 14d, 30d
  
  // 根据用户表现调整
  adjustInterval(performance: number): number {
    // performance: 0-1 (答题正确率)
    if (performance >= 0.9) return nextInterval * 1.5;  // 加速
    if (performance >= 0.7) return nextInterval;        // 保持
    if (performance >= 0.5) return nextInterval * 0.5;  // 减速
    return previousInterval;  // 重来
  }
}
```

## 10. 数据驱动运营系统

### 10.1 数据埋点策略

游戏化学习的核心在于**数据驱动优化**。我们需要追踪用户的每一个关键行为，分析学习路径，找出卡点，持续优化体验。

```
┌─────────────────────────────────────────────────────────────────────┐
│                    DATA-DRIVEN LEARNING OPTIMIZATION                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   用户行为                分析维度                优化动作          │
│   ─────────              ─────────              ─────────          │
│                                                                     │
│   🎮 答题               📊 正确率分析            🔧 调整难度         │
│   ⏱️ 停留时间           📊 时间分析              🔧 优化引导         │
```
